# Тест-кейсы для Avito API

## 1. Создание объявления

| ID | Название | Шаги | Ожидаемый результат | Фактический результат |
|----|----------|------|-------------------|--------------------|
| TC-01 | Создание объявления с валидным sellerID | 1. Сгенерировать sellerID в диапазоне 111111–999999<br>2. Отправить POST /item с валидными данными | Код ответа 200, в ответе есть `id`, `name`, `price`, `statistics` | Пройден |
| TC-02 | Создание объявления без sellerID | 1. Отправить POST /item без sellerID | Код ответа 400 | Пройден |

## 2. Получение объявления

| ID | Название | Шаги | Ожидаемый результат | Фактический результат |
|----|----------|------|-------------------|--------------------|
| TC-03 | Получение существующего объявления | 1. Создать объявление<br>2. GET /item/{id} | Код ответа 200, ответ содержит `id`, `name`, `price`, `statistics` | Пройден |
| TC-04 | Получение несуществующего объявления | GET /item/{randomIdNotExist} | Код ответа 404 или 400 | Пройден |

## 3. Получение всех объявлений продавца

| ID | Название | Шаги | Ожидаемый результат | Фактический результат |
|----|----------|------|-------------------|--------------------|
| TC-05 | Получение всех объявлений существующего продавца | 1. Создать несколько объявлений для одного sellerID<br>2. GET /{sellerID}/item | Код ответа 200, все объявления принадлежат указанному sellerID | Пройден |
| TC-06 | Получение объявлений несуществующего продавца | GET /{randomSellerIDNotExist}/item | Код ответа 200 (пустой массив) или 404 | Пройден |

## 4. Получение статистики

| ID | Название | Шаги | Ожидаемый результат | Фактический результат |
|----|----------|------|-------------------|--------------------|
| TC-07 | Получение статистики существующего объявления | 1. Создать объявление<br>2. GET /statistic/{id} | Код ответа 200, в ответе есть `likes`, `viewCount`, `contacts` | Пройден |
| TC-08 | Получение статистики несуществующего объявления | GET /statistic/{randomIdNotExist} | Код ответа 404 или 400 | Пройден |

## 5. Нефункциональные проверки

| ID | Название | Шаги | Ожидаемый результат | Фактический результат |
|----|----------|------|-------------------|--------------------|
| TC-09 | Время отклика сервера | POST /item с валидными данными | Время отклика < 10 секунд | Пройден |
